# Seymour's NNF Mods
Mods created for Nubby's Number Factory. Intended to be used with skirlez' g3man launcher.

## Forgery Mods
*These mods require the Nubby's Forgery mod to be installed.*

* Old Glass Tony
 * Brings back the old Glass Tony as a separate supervisor.
* Old Balogna Tony
 * Brings back the old Balogna Tony as a separate supervisor.
* Wheelie Coffin Item
 * Adds the Wheelie Coffin item.
